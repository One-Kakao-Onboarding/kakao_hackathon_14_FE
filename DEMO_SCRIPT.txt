------------------------------------------
[화면: Hero Banner 보이기]

"1인 가구의 인테리어, 어디서부터 시작해야 할지 막막하셨나요?

맞춤형 인테리어와 가구를 추천해주는 서비스입니다."


[00:15-00:45] 튜토리얼 소개 (30초)
------------------------------------------
[화면: Tutorial Carousel - 4개 슬라이드 넘기며]

"카카오 홈즈는 4단계로 이루어져 있습니다.

첫째, 방 사진 한 장으로 AI가 인테리어를 재해석합니다.

둘째, 월세 거주자를 위한 무타공 가구를 우선 추천합니다.

셋째, 카카오톡으로 친구들에게 투표를 받을 수 있습니다.

넷째, 이사일에 맞춰 선택한 가구를 한 번에 배송받습니다.

지금부터 실제로 사용해보겠습니다."


[00:45-02:45] 메인 시연 (120초)
------------------------------------------

## 1단계: 이미지 업로드 & 영역 선택 (30초)
[화면: Upload Section으로 스크롤]

"먼저 현재 내 방의 사진을 업로드합니다.
[이미지 업로드 클릭 → 파일 선택]

사진이 업로드되면, '영역 선택하기' 버튼을 누릅니다.
[영역 선택하기 버튼 클릭]

변경하고 싶은 영역을 드래그로 선택합니다.
[화면에 원 그리기 - 소파나 가구 영역]

영역 선택이 완료되면 '편집 완료'를 클릭합니다.
[편집 완료 버튼 클릭]"


## 2단계: AI 분석 & 설정 (30초)
[화면: AI 분석 중 모달 → 자동으로 Settings로 전환]

"AI가 이미지를 분석하고 있습니다.
[1.5초 대기 - 모달 보이기]

분석이 완료되면, AI가 자동으로 어울리는 무드를 추천해줍니다.
[Settings 화면 - AI 추천 배지가 있는 3개 무드 강조]

모던, 미니멀, 우드 스타일이 추천되었습니다.
원하는 무드를 자유롭게 선택할 수 있습니다.
[클릭으로 선택/해제 시연]

주거 형태도 선택합니다. 월세를 선택하면...
[월세 클릭]

무타공 제품 위주로 추천받을 수 있습니다.
[다음 단계로 버튼 클릭]"


## 3단계: AI 인테리어 생성 & 결과 (30초)
[화면: Ready 화면]

"설정이 완료되었습니다.
선택한 스타일과 주거 형태가 표시됩니다.
[오른쪽 패널 - 설정 정보 보이기]

이제 'AI 인테리어 생성하기' 버튼을 클릭합니다.
[생성 버튼 클릭 → 로딩]

AI가 선택한 영역에 새로운 인테리어를 적용하고 있습니다.
[10-30초 대기 또는 빠르게 편집]

생성이 완료되었습니다!
[Result 화면 - Before/After 슬라이더]

슬라이더를 움직여 원본과 AI 결과를 비교할 수 있습니다.
[슬라이더 좌우로 움직이며 비교]"


## 4단계: 상품 추천 & 투표 (30초)
[화면: Result 화면 → Products로 전환]

"마음에 드시나요? 그렇다면 이제 가구를 선택해볼까요.

'AI 추천 상품 보기' 버튼을 클릭합니다.
[AI 추천 상품 보기 버튼 클릭]

AI가 선택한 스타일에 맞는 4개의 가구를 추천해줍니다.
[Products 화면 - 4개 상품 카드 보이기]

각 상품에는 무타공, 이동 가능 등의 특징이 표시됩니다.
월세 거주자에게 최적화된 상품들입니다.

투표를 위해 2개의 상품을 선택합니다.
[2개 상품 클릭 - 체크 표시 강조]

2개 선택이 완료되면, '친구들에게 투표 올리기' 버튼이 활성화됩니다.
[버튼 클릭]

카카오톡으로 친구들에게 공유하여 의견을 받을 수 있습니다.
[알럿 또는 공유 화면 보이기]"


[02:45-03:00] 아웃트로 (15초)
------------------------------------------
[화면: 전체 페이지 스크롤 또는 Hero Banner로 복귀]

"카카오 홈즈는 AI 기술로 1인 가구의 인테리어 고민을 해결하고,
주거 형태에 맞는 맞춤형 가구를 추천하며,
소셜 투표를 통해 더 나은 선택을 돕습니다.

🎬 촬영 팁
========================================

1. 사전 준비
   - 이미지 파일 미리 준비 (방 사진)
   - 브라우저 탭 정리 (데모 페이지만)
   - 화면 해상도 1920x1080 설정
   - 마우스 커서 강조 프로그램 사용 권장

2. 녹화 설정
   - OBS Studio 또는 Loom 사용
   - 1920x1080 해상도
   - 60fps 권장
   - 마이크 테스트 필수

3. 촬영 순서
   ① 스크립트 리허설 2-3회
   ② 타이머 보면서 시간 체크
   ③ 본 녹화 (여유있게 3-4회 촬영)
   ④ 가장 자연스러운 테이크 선택

4. 편집 포인트
   - AI 생성 대기 시간은 빠르게 편집
   - 중요 UI 요소에 줌인/하이라이트
   - 배경 음악 추가 (저작권 free)
   - 자막 추가 (핵심 키워드)

5. 강조할 핵심 가치
   ✓ AI 자동 분석 & 추천
   ✓ 무타공 제품 (월세 거주자 배려)
   ✓ 소셜 투표 (친구 의견 반영)
   ✓ 통합 배송 (이사일 맞춤)
   ✓ 원스톱 솔루션 (한 페이지에서 완결)
